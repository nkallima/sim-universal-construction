Detailed Description

This file exposes the API of a simple re-entrant synchronization barrier. This implementation is a classic two phase (arrive and leave phases) re-entrant barrier, i.e. arrive and leave phases. Examples of usage could be found in almost all the provided benchmarks under the benchmarks directory.

Data Structure Documentation

◆ SynchBarrier

struct SynchBarrier

Barrier stores the state of an instance of the a backoff object. Barrier should be initialized using the BarrierSet function.

Data Fields
volatile int32_t arrive This integer is atomically decreased by one whenever a thread enters the arrive phase of the barrier.
volatile int32_t leave This integer is atomically decreased by one whenever a thread enters the leave phase of the barrier.
volatile bool arrive_flag This variable is set to true if all the n threads have entered to the arrive phase.
volatile bool leave_flag This variable is set to true if all the n threads have entered to the leave phase.
volatile int32_t val_at_set The initial value set to arrive and leave variables.
